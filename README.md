# Week 4 Worksheet Repository: Exploratory Data Analysis

This repository is for the Timothy Tran's ECC3479

## Repository Structure

- `README.md`: this file
- `requirements.txt`: Python packages for the activity
- `data:raw/`: Folder with all raw datasets that has been untouched
- `data:clean/`: Folder with data sets that has been cleaned and checked for errors
- `code/Code.ipynb`: notebook with code
- `output/`: save any figures or exported outputs here

## Getting Started

1. Create and activate a Python environment.
2. Install the required packages with `pip install -r requirements.txt`.
3. Open `code/Code.ipynb` in VS Code.
4. Work through the notebook and save any figures you want to keep in `output/`.
